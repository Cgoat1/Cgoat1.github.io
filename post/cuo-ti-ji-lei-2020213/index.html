错题1

(1997年学习报竞赛题)由1997盏亮着的电灯，各由一个拉线开关控制着，先按其顺序编号为1,2,⋯ ,19971,2,\cdots,19971,2,⋯,1997，然后将编号为2的倍数的灯线拉一下；再将编号为3的倍数的灯线拉一下；最后将编号为5的倍数的灯线拉一下三次拉完后亮着的灯的盏数为        ‾\underline{\ \ \ \ \ \ \ \ }        ​

错误解法

1997÷2=998⋯11997\div 2=998\cdots 11997÷2=998⋯1
1997÷3=665⋯21997\div 3=665\cdots 21997÷3=665⋯2
1997÷5=399⋯21997\div 5=399\cdots 21997÷5=399⋯2
1997÷(2⋅3)=332⋯51997\div (2\cdot 3)=332\cdots 51997÷(2⋅3)=332⋯5
1997÷(3⋅5)=133⋯21997\div (3\cdot 5)=133\cdots 21997÷(3⋅5)=133⋯2
1997÷(2⋅5)=199⋯71997\div (2\cdot 5)=199\cdots 71997÷(2⋅5)=199⋯7
1997÷(2⋅3⋅5)=66⋯171997\div (2\cdot 3\cdot 5)=66\cdots 171997÷(2⋅3⋅5)=66⋯17
1997−998−665−399+332+133+199−66=5331997-998-665-399+332+133+199-66=5331997−998−665−399+332+133+199−66=533

错误原因

容斥时重复和漏加区域

正确解法

999
